    Onbeperkt aantal pogingen toegestaan

    Voor dat je je webpagina kan gaan bouwen moet je weten wat er allemaal op komt te staan. In deze opdracht gaan we de eerste ruwe opzet maken dit kan je als je meer ideen krijgt later aanvullen. Het document moet wel gereed zijn voor je je webpagina gaat bouwen zodat je een duidelijk beeld hebt over wat je gaat bouwen.

    Naast de teksten die je wil neerzetten op je website, kun je ook alvast bedenken welke afbeeldingen je wilt laten zien (en wanneer je deze wilt maken) en een ruwe schets maken van de lay-out.

    Hieronder staat een voorbeeld van een ruwe schets.

    Opdracht

    • Maak een document voor je ontwerp en bewaar deze op een logische plek op je computer (in dezelfde folder als je logboek).
    • Schrijf in dit document wat je moet laten zien op je webpagina bedenk ook alvast hoe je dit gaat verzamelen.
    • Maak alvast een (ruwe schets) van de lay-out, dit kun je doen op papier of in PowerPoint of een ander programma (zie voorbeeld hierboven).
    • Op je webpagina moet je straks een screenshot van je ruwe schets op je webpagina plaatsen.
    • Het is belangrijk dat je elke keer als je iets doet je logboek bestand bijwerkt, doe dit dus niet pas aan het einde van het blok.
